# 🤖 ZyraCosmicAI Telegram Bot

A hyper-realistic AI girl chatbot with voice interaction, emotions, and human-like behavior.
Built to feel real, not robotic.

---

✨ Features

- 💬 Human-like Chatting
  
  - Short, natural Hinglish replies
  - Mood-based responses (happy, sad, bored, etc.)
  - Relationship-based behavior (stranger → close)

- 🎭 Dynamic Personality
  
  - Acts like a real 17-year-old girl
  - Mood swings, selective replies, real texting style

- 🎤 Voice System
  
  - Voice message → speech-to-text
  - AI reply → realistic voice output
  - Toggle voice/text modes

- 🧠 Emotion Detection
  
  - Detects user mood
  - Adapts replies accordingly

- ⚡ Smart Behavior
  
  - Random delays (human feel)
  - Sometimes ignores messages (realistic)
  - Group chat awareness

---

🚀 Commands

Command| Description
"/start"| Start chatting
"/help"| Show help
"/mood"| Check Zyra's mood
"/voiceon"| Enable voice replies
"/voiceoff"| Disable voice replies
"/clear"| Reset conversation

---

🛠️ Tech Stack

- Python
- python-telegram-bot
- OpenAI API (GPT + Whisper + TTS)
- gTTS (fallback voice)

---

📦 Installation

git clone https://github.com/your-username/your-repo.git
cd your-repo
pip install -r requirements.txt

---

⚙️ Environment Variables

Create a ".env" file or set variables:

TELEGRAM_BOT_TOKEN=your_token
OPENAI_API_KEY=your_openai_key
ADMIN_ID=your_telegram_id

---

▶️ Run Locally

python bot.py

---

🌐 Deployment (Render)

1. Push code to GitHub
2. Create Web Service on Render
3. Add environment variables
4. Start command:
   python bot.py
5. (Optional) Use uptime monitor to keep alive
